Current projects
  • Drone Recognition and Delivery Challenge

    This competition focuses on real-world drone operations within a complex obstacle-laden environment. Autonomously flying drones must navigate through obstacles, recognize targets, and deliver payloads accurately. The event is set in a simulated urban area, where drones start with onboard cargo, maneuver through a designated path, locate specific delivery points by recognizing targets, and complete the delivery before autonomously landing.

    Technical Challenges:

    • Autonomous Flight: Drones must take off, fly, and land without any manual control, demonstrating advanced flight control algorithms.
    • Obstacle Avoidance: Drones navigate through a complex environment with physical barriers, requiring real-time path planning and obstacle detection.
    • Target Recognition: Drones need to identify specific delivery points using visual recognition systems, which are crucial for accurate payload deployment.
    • Payload Delivery: Drones must precisely drop cargo at targeted spots, testing their precision and stability in payload handling.

  • Multi-rotor UAV Swarm Collaborative Search

    This competition centers on a technically demanding challenge conducted within a computer-simulated urban environment. Set in a virtual cityscape, teams deploy a swarm of drones on a critical search mission. The primary objective is to utilize the swarm to efficiently locate and identify several terrorists concealed throughout the urban setting, all within a designated timeframe. Each drone in the swarm communicates using ROS topics, enabling seamless real-time data exchange and coordination, thereby enhancing the effectiveness of the collective search effort.

    Technical Challenges:

    • Target Detection: Identifying terrorists from other civilians and objects in the environment.
    • Vision-Based Servoing: Using cameras to navigate and guide drones towards targets.
    • Motion Planning: Strategically planning the paths of drones to optimize search efficiency.
    • Multi-Drone Coordination: Ensuring drones operate in harmony, sharing tasks and data to effectively complete the search mission.
